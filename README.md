# AppointDesk

AppointDesk is a Django-based appointment management system designed for receptionists and businesses to manage appointments efficiently.

## Features
- User Authentication
- Add, Update & Delete Appointments
- Search Functionality
- Appointment Status Management
- Django Admin Panel
- Responsive Bootstrap UI

## Technologies Used
- Python
- Django
- HTML
- Bootstrap 5
- MySQL
- Crispy Forms

## Run Project

```bash
python manage.py runserver
